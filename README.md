# rawthumbnailer
Show thumbnails for raw files in Debian Jessie

Package for Debian 8.0 (Jessie). It shows thumbnails for cr2, crw, mrw, nef, pef, rw2, srw, orf, dng, dcraw, raf, dcr, k25, kdc, raw, x3f, arw, sr2 and srf files in gnome.
